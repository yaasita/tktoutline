* これは何？
文章をアウトライン表示します
pukiwikiに最適化しました

* インストール
vundle等で管理されている場合は
 Bundle 'yaasita/tktoutline'
をvimrcに追加して以下のコマンドでインストールされます
 :BundleInstall

手動で入れる場合は~/.vimにコピーします

* 使い方
以下のコマンドで実行現在開いているバッファのアウトラインが表示されるはずです
 :TktOutline

* ヘルプ
 :h tktoutline.txt@ja

* その他
g!//dとs///で整形しているだけなので、他のwikiとか文章にも使えるかも・・・
